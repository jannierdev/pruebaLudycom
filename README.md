
## Backend

Instalamos dependencias

```bash
  cd api-rest/server && npm install
```

Corremos el nodemon

```bash
  npm run build
```

En otra consola corremos el proyecto

```bash
  npm run dev
```

## Frontend

Instalamos dependencias

```bash
  cd front-end && npm install
```

Corremos el proyecto

```bash
  ng serve
```
